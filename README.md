# EstateBot

EstateBot — это Telegram-бот для поиска недвижимости в Болгарии, Румынии и Сербии.

## Ссылки

- **Бот:** https://t.me/europe_estate_bot
- **Сайт:** https://dronov-dmitry.github.io/estate-bot/

## Возможности

- **Умный поиск** — фильтры по цене, цене за м², площади и городу
- **Ежедневные обновления** — новые объявления приходят автоматически
- **Премиум-режим** — больше объявлений и приоритетные уведомления
- **Размещение объявлений** — команда `/advert` для вашего объекта
- **3 страны** — актуальные предложения на популярных рынках Балкан
